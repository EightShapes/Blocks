h1. EightShapes Blocks

EightShapes Blocks is a toolkit to organize, annotate, and modularize HTML prototypes created by <a href="http://eightshapes.com">EightShapes, LLC (http://www.eightshapes.com)</a>.

h2. Quickstart

To get going with Blocks you can:

* <a href="https://github.com/EightShapes/Blocks/releases/latest">Download the latest release</a>
* <a href="http://bower.io/">Install with Bower</a>: bower install EightShapes-Blocks

h2. Documentation

The functionality of Blocks is documented in the <a href="https://github.com/EightShapes/Blocks/wiki">wiki</a> of this repository.
