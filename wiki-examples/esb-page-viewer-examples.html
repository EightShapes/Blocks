ESB Page Viewer Examples

Default Example

This is the simplest example

				
<div data-esb-page-viewer="http://example.com"></div>

Responsive Example

This is how to display the same page at multiple viewports to get a side-by-side responsive comparison.

				
<div data-esb-page-viewer="http://eightshapes.com" data-esb-viewport-width="320" data-esb-scale="0.25" data-esb-title="iPhone"></div>
<div data-esb-page-viewer="http://eightshapes.com" data-esb-viewport-width="768" data-esb-scale="0.25" data-esb-title="iPad"></div>
<div data-esb-page-viewer="http://eightshapes.com" data-esb-viewport-width="1600" data-esb-scale="0.25" data-esb-title="Desktop"></div>

Gallery Example

This is how to display multiple pages side-by-side

				
<div data-esb-page-viewer="http://apple.com/home"></div>
<div data-esb-page-viewer="http://apple.com/mac"></div>
<div data-esb-page-viewer="http://apple.com/iphone"></div>
<div data-esb-page-viewer="http://apple.com/ipad"></div>
<div data-esb-page-viewer="http://apple.com/watch"></div>

On Page Config Example

Use on page config to wrap certain viewers with options that you don't want to apply to other viewers

				
<h4>280px wide</h4>
<div data-esb-page-viewer-config="" data-esb-width="280">
	<div data-esb-page-viewer="http://css-tricks.com"></div>
	<div data-esb-page-viewer="http://css-tricks.com/video-screencasts"></div>
	<div data-esb-page-viewer="http://css-tricks.com/almanac"></div>
	<div data-esb-page-viewer="http://css-tricks.com/snippets"></div>
</div>
