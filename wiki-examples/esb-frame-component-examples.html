ESB Frame Component Examples

The following examples show how Frame can be used in conjunction with Component Loader.

Loading a Component in a Frame

To frame a component, specify the component name for the data-esb-frame attribute and add a data-esb-variation attribute to specify the variation.

By default components are rendered at 20% scale. The Frame will be sized to fit the component's width and height, though the default Frame CSS specifies a min-width of 200px for the overall Frame wrap. 

						
<div data-esb-frame="newsletter-subscription" data-esb-variation="default"></div>

Scaling a Frame Component

To change a component's scale use the data-esb-scale attribute. 

						
<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="default" 
     data-esb-scale="1"></div>

Changing the viewport

If you need to demonstrate responsive behavior of a component you can change the viewport of the frame using the data-viewport-width attribute.

In this example the button on the newsletter sign up form goes full width and the header reduces its font size at 320px and below. 

						
<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="default" 
     data-esb-viewport-width="320" 
     data-esb-scale="1"></div>

Cropping a component

If you're displaying a gallery of components and would like small "thumbnail" style Frames of each component you can use cropping to show just a relevant portion of each component.

Cropping works by setting the data-esb-viewport-width to a value higher than data-esb-width and setting data-esb-crop="true". You can also set data-esb-height to create the exact dimensions you want.

NOTE: data-esb-crop="true" causes the dimensions annotation to be hidden automatically 

						
<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="default" 
     data-esb-viewport-width="400" 
     data-esb-width="150" 
     data-esb-height="150" 
     data-esb-scale="1"></div>

Zooming a cropped component

To zoom in on a cropped section of an element set data-esb-scale to a value greater than 1 

						
<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="default" 
     data-esb-viewport-width="400" 
     data-esb-width="150" 
     data-esb-height="150" 
     data-esb-scale="2"></div>

Crop Positioning

You can change which part of the component is visible in a cropped component by using the data-esb-crop-offset-x and data-esb-crop-offset-y attributes. The value passed to data-esb-crop-offset-x will cause the component to be shifted to the left that many pixels. The value passed to data-esb-crop-offset-y will cause the component to be shifted to the top that many pixels. 

						
<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="default" 
     data-esb-viewport-width="400" 
     data-esb-width="150" 
     data-esb-height="150" 
     data-esb-offset-x="100" 
     data-esb-offset-y="480" 
     data-esb-scale="2"></div>
