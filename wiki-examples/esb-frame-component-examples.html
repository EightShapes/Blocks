ESB Frame Component Examples

The following examples show how Frame can be used in conjunction with Component Loader.

Loading a Component in a Frame

To frame a component, specify the component name for the data-esb-frame attribute and add a data-esb-variation attribute to specify the variation.

Framing a component inherits all of Frame's defaults which means with the minimum amount of options you'll see your component rendered in a 1000px viewport at 20% scale. Probably not what you want. 

						
<div data-esb-frame="newsletter-subscription" data-esb-variation="default"></div>
