Showing/Hiding Marks

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Illum maiores numquam sed iusto vero, suscipit facilis.

All marks on a page can be shown/hidden by firing custom javascript events. To hide all marks trigger a hide-all-esb-marks event. To show all marks, trigger a show-all-esb-marks event. 
						
//jquery Hide
$(document).trigger("hide-all-esb-marks");

//jquery Show
$(document).trigger("show-all-esb-marks");

//Javascript Hide
if (window.CustomEvent) {
    var event = new CustomEvent('hide-all-esb-marks');
} else {
    var event = document.createEvent('CustomEvent');
    event.initCustomEvent('hide-all-esb-marks', true, true);
}
document.dispatchEvent(event);

//Javascript Show
if (window.CustomEvent) {
    var event = new CustomEvent('show-all-esb-marks');
} else {
    var event = document.createEvent('CustomEvent');
    event.initCustomEvent('show-all-esb-marks', true, true);
}
document.dispatchEvent(event);
